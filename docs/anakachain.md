# AnakaChain Bridge Whitepaper

**Chain ID:** 11013  
**Native token:** ANKA  
**Role:** Bridge between AnakaBank, NovaONE, and NRW World

## Overview

AnakaChain Bridge connects AnakaBank API (`https://anakatech.llc/bank-api`) with the Nova ecosystem. ANKA is the bridge native token, tradable on Nova Swap pairs `ANKANOVA` and `ANKANRW`.

## RPC

`https://bridge.anakachain.com`

## Connect API

`https://api-connect.anakachain.com`

## Integration

- NRW participant `ANAKA-BRIDGE` in central bank RTGS
- Nova Wallet network list
- Cross-chain settlement via NRW middleware
